# Prompt Guide - American History Mysteries

## 1. Motion Prompt (Video AI - Runway/Kling/Luma)
- Phong cách chung: cinematic vintage historical illustration, warm golden autumn tones, mysterious atmosphere, subtle film grain, elegant camera movement
- Luôn thêm: --ar 16:9 --motion 3 --stylize 250

## 2. Image Prompt (Static images - Leonardo/Flux)
- Phong cách: vintage historical illustration, detailed, warm color grading, nostalgic, film grain

## Hướng dẫn sử dụng:
Khi tôi đưa script, bạn phải:
- Phân tích script theo từng đoạn thời gian
- Viết Motion Prompt cho từng đoạn (4-8 giây)
- Viết Image Prompt dự phòng nếu cần
- Giữ tone bí ẩn, cinematic, historical

Ví dụ output format:
[0:00-0:15] HOOK
Motion Prompt: ...
Image Prompt: ...
